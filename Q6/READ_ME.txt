There are two path in the code.cpp:
g_read_xml_path: this file contains the path of each photo. 
g_save_xml_path: result saved in this file, it contains the camera matrix and the distCoeffs matrix.

If you want run the code, change the g_read_xml_path and g_save_xml_path to your own path. 
Then add your photo path to the left_photo_list.xml
